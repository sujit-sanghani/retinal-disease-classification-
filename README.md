# 🧠 Retinal Disease Classification

## 🚀 Overview
This project is a deep learning-based system designed to classify retinal diseases from medical images. It aims to assist in early detection and diagnosis by analyzing retinal scans using convolutional neural networks (CNN).

## 🎯 Objective
To build an intelligent system that can accurately identify retinal diseases and support healthcare professionals with faster diagnosis.

## 🛠️ Tech Stack
- Python  
- TensorFlow / Keras  
- OpenCV  
- NumPy / Pandas  

## ⚙️ Features
- Image preprocessing and enhancement  
- Deep learning model for classification  
- Disease prediction from retinal images  
- Easy-to-use interface for testing  

## 🧪 How It Works
1. Input retinal image  
2. Preprocessing (resize, normalize, enhance)  
3. Model predicts disease class  
4. Output result with prediction
